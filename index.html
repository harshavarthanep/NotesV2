    💡 How ZenDocs works

    Writing & autosave

    Just type — every change saves automatically (“Saving… → Saved” in the header). Nothing is written to the cloud when you merely open or read a note. The footer shows live word count, characters and reading time.

    Power moves while typing

    • Type / on a line → command menu: /h1 /h2 /h3 headings, • bullets, 1. numbered, ☑ checklist, ❝ quote, {} code, 📅 today’s date, 🕐 time.
    • Type [[ → pick or create a linked note.
    • Type # → existing tags autocomplete.
    • Paste or type a URL → it turns into a blue clickable link automatically.
    • The toolbar image button inserts photos — they’re compressed automatically (max 1000px) so notes stay fast and within sync limits.
    • Navigate every popup with ↑↓, insert with Enter or Tab.
    • The footer shows live writing speed, flow time and task completion for the open note (large screens).

    Links (step by step)

    1. Highlight the words you want to link.
    2. Tap the link button 🔗 in the toolbar.
    3. Enter the URL → Apply. The text becomes the hyperlink.
    4. Click any link to Open / change / Remove it (remove keeps the plain text).
    5. Ctrl+Click opens it straight in a new tab.

    Sidebar & organizing

    • Folders are tiles now — icon on top, name below (Windows-style); hover/touch shows rename & delete on the tile corners; the badge counts notes inside.
    • ▾ arrows next to Folders / # Tags / Recent collapse each section for a minimal look (remembered).
    • Folders: ✏️ renames, × deletes (notes are kept). Double-tap the name also renames.
    • #tag rows filter your notes; tap again for all. Ctrl+K searches everything — titles, content, even #tag queries.
    • Every note row: pin (keeps important notes at the top), info (dates, tags, links, word stats, share status) and duplicate — always visible on touch screens.

    Notes panel

    • Comments: select text → speech-bubble → write → Resolve removes note + highlight.
    • Outline: your headings as a page navigator — the section you’re reading is highlighted, ▾ collapses sub-sections, and the search box filters headings (like Obsidian’s page nav).
    • Linked mentions: every note that links here via [[…]].
    • Unlinked mentions (discovery): notes that mention this note’s title but don’t link to it yet — open one and wrap the mention in [[…]] to connect your knowledge. Runs on notes already in memory, costing zero extra database reads.

    Graph view (Ctrl+G)

    • Hold or hover a node to spotlight it — its connections light up in your accent color while everything else dims (Obsidian-style).
    • Node size = importance: the more notes link to it, the bigger the dot. Notes are tinted by folder so clusters are visible at a glance.
    • Drag any dot or its name to arrange; drag empty space to pan.
    • Pinch with two fingers, scroll, or use + / − / Reset to zoom; the graph auto-fits on open so everything is visible.
    • Tap a note to open it, an accent #tag to filter by it.

    New in V3.8

    • Local AI (sparkles icon): on-device summary + suggested #tags — instant, private, no cloud, no downloads.
    • Study mode (cap icon): auto-flashcards from your headings, bold terms and lists, plus a step-by-step Guided walkthrough.
    • Collapsible sections: a ▸ chevron floats beside every heading — click to fold/unfold that whole section (Notion-style toggles).
    • Task progress bar appears above the editor whenever the note has ☐ tasks.
    • Time machine (clock icon): local snapshots of this note as you write — drag the slider, preview, restore (undo-able). Stored only on this device.
    • Deep Think (grid icon, desktop): a live structural map of your note — sections sized by content, links and task counts, click to jump.
    • Canvas frames: “+ Frame” groups cards — drag the frame and everything inside moves together; resize from the corner, double-tap the label to rename.
    • Editor zoom (A−/A+ or the mobile menu) · images: tap (mobile) or double-click (desktop) any image for a full-screen viewer with pinch/zoom and download.
    • Task analytics on Home: a 7-day completion chart, logged locally.

    Install, features & reminders (V3.9)

    • Install ZenDocs: a popup offers installation on Windows, Android and iOS (on iPhone: Share → Add to Home Screen). Installed apps get full screen and an offline shell.
    • Features on this device (sliders icon in the sidebar footer): switch any feature off to hide it on this device only — it never syncs, and an amber dot quietly reminds you you’re not running the full app. “Reset to default” turns everything back on.
    • Reminders (bell, in the ⋯ and ☰ menus): pick a date & time for any note. You’ll get a notification, a toast and a vibration (on phones). Reminders are stored on this device and fire while ZenDocs is open or installed.

    Home dashboard

    The Home button (first in the views row) aggregates your workspace: pinned & starred notes, all open tasks (kanban To-do / In-progress plus every unchecked ☐ item across your notes), today’s daily note, and recent activity — tap anything to jump there.

    Move blocks (Notion-style)

    On desktop, hover any paragraph, heading or list item — a small ▲▼ handle appears on the left. Click to move that block up or down; undo works normally.

    Daily notes & Calendar

    1. Today in the sidebar opens (or creates) today’s note — with priorities ☑, notes and a reflection prompt.
    2. Month opens the calendar; ☀️ = day has a daily note, the badge counts notes created that day.
    3. Tap a day → panel with that day’s notes, each with ℹ️ info and ⧉ duplicate, plus “Open/Create daily note”.
    4. Tap the month name → month & year dropdowns (scroll any year, 1950–2100).

    Kanban board

    • Drag cards between To do / In progress / Done (◀▶ buttons on mobile); columns fill the screen on desktop and scroll when full.
    • + adds an existing note; Aa opens the card composer — heading + multi-line content with # heading, - bullets and [] tasks; every card is a real note, so open it for full formatting.
    • Drag the corner handle to grow a card’s content area (remembered). ✕ asks before removing (the note is kept).

    Canvas (brainstorming board)

    1. Double-click empty space (or the + bubble on mobile) → new idea card, ready to type.
    2. + Note pins an existing note to the board.
    3. Drag cards anywhere; drag the background to pan; + / − / Fit or scroll to zoom.
    4. Card tools (icons on each card): edit · color · duplicate · connect two cards · delete · open note.
    5. Saves automatically — only when something actually changed.

    Templates

    Use a built-in (Meeting, Project, Cornell, Weekly review) or open any note → “Save current note as template”. Your templates support rename and delete (pencil / bin icons), and now dynamic variables: {{date}}, {{time}}, {{datetime}}, {{day}}, {{title}} and {{clipboard}} are filled in automatically when the template is used.

    Export

    Word (.docx) — a real Word file with formatting intact · PDF — direct paginated A4 download · Markdown (.md) — Obsidian-ready with #tags and [[links]] · Plain text · Copy as Markdown puts the note on your clipboard for pasting into Obsidian. Export engines load only when you use them, so the app stays light.

    Appearance

    The Theme button in the sidebar footer opens solid accents and gradient themes (Ocean, Sunset, Aurora, Candy, Ember) that recolor tags, buttons, highlights and headers across the app — plus a custom color picker (dashed swatch — choose any color) and the light/dark toggle. The editor caret follows your accent in both light and dark mode. Light mode has a warm, paper-like feel including the sidebar. Remembered per device.

    Shortcuts

    Ctrl+K search · Ctrl+G graph · Ctrl+E reading mode · Ctrl+. zen mode · Ctrl+S save / go online · Ctrl+Z / Y undo & redo · Esc closes any view.

    Offline

    The cloud icon in the footer pauses cloud sync (great for flights); edits are kept and Ctrl+S reconnects and syncs everything. After a minute offline a small floating reminder appears in the corner — tap Sync now or dismiss it; it never covers your writing.
